    Clase para gestionar colecciones de objetos de tipo StarBucks

    Esta clase nos permite almacenar e iterar sobre colecciones de objetos de tipo StarBucks. Se usa normalmente para devolver conjuntos de resultados de la Base de Datos de forma que se pueda iterar sobre ellos fácilmente. Implementa la interfaz Iterator de PHP

    author Miguel A. Pedregosa
    since 1.0
